The Vault is my project to house, upkeep, and improve my knowledge base across different fields. 

So far we have: 
- Integrated Local LLMs like Gemini and DeepSeek to be able to generate mock exams, quizzes and flashcards based on the database, reformat, reword and restructure notes. 
- Setup a version management system using GitHub, as well as a cloud backup 
- Established a Home Page with the relevant resources, such as this roadmap
- Kickstarted the vaults knowledge base using my first test as reference 
- Established basic formatting guidelines, and stylebook

Recently: 
- Establish a Glossary, Auto Glossary Command
- epub to txt and txt to audio
- established inline glossary etiquette for short definitions to avoid note clutter, where small definitions such as prone and supine will be relegated to existing in the glossary with the title :: definition ^tag format, allowing it to be linked and referenced properly. This also allows a definition to exist as its own card such as diffusion and still have its glossary definition

We still need to:

- Common CS styling and JS scripts we might use 
- Establish an ideal length for notes, and ideal # of connections 
- Better Integration with Drawings, Images and Charts
