**Personal Protective Equipment (PPE)** consists of equipment worn to minimize exposure to biological, chemical, physical, and environmental hazards, in accordance with **[[BSI]] and [[Standard Precautions]]**.

| Universal PPE | Situational PPE |
| ------------- | --------------- |
| Gloves        | Gown            |
| Boots         | Goggles         |
| Face Mask     | Reflective Vest |
|               | Body Armor      |
|               | Respirator      |
PPE is selected right after a call is received, and may be reevaluated as needed. If there is a questions whether you would need certain protective gear, use it.
